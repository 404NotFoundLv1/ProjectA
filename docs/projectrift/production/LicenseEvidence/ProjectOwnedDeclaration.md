# Project-Owned ProjectRift Placeholder and Data Content

This declaration covers the ProjectRift-native code, data assets, graybox maps, UI placeholders, and placeholder feedback content currently under `/Game/ProjectRift` as recorded in the v0.9.0 registry.

It does not cover future imported third-party meshes, textures, animations, audio, fonts, or generated media. Each such production asset must have an exact registry entry and its own retained license/invoice evidence before use.
