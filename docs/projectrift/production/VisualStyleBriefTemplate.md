# ProjectRift Visual Style Brief Template

This is a one-page human-authored template. It intentionally contains no generated visual material.

- **Chapter / location:**
- **Narrative mood:**
- **Primary palette:** list hue, value range, and accessibility contrast target.
- **Secondary/accent palette:**
- **Lighting direction:** key, fill, rim, fog/exposure limits, and forbidden conditions.
- **Material language:** surface age, roughness/metalness range, decal density, and readable silhouettes.
- **Gameplay readability:** enemy, interactable, extraction, hazard, ally, and Boss color separation.
- **Reference board:** attach only reviewed images with source and license evidence.
- **Performance constraints:** expected streaming partitions, high-cost materials, VFX density, and shadow budget.
- **Approvals:** art owner, technical art owner, license reviewer, date.
